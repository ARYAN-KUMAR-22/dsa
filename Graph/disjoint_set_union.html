Current Insight

A disjoint set keeps track of which elements belong together while making repeated connectivity checks very fast.

Current Groups

What DSU Does

  • Disjoint Set Union tracks which elements belong to the same connected component.
  • find(x) returns the representative root of the set containing x.
  • union(a, b) merges two sets when their roots are different.

Why Path Compression Matters

  • Each find walks parent links until it reaches a root.
  • Path compression rewires visited nodes directly to that root.
  • Future find calls become much shorter after compression happens.

Why Union By Size Helps

  • Always attaching the smaller tree under the larger root avoids tall chains.
  • That keeps the forest shallow and works especially well with path compression.
  • Together they make DSU extremely efficient in practice.

Where DSU Is Used

  • Cycle detection in undirected graphs.
  • Kruskal's minimum spanning tree algorithm.
  • Dynamic grouping, connectivity tracking, and offline query problems.

Pseudocode

makeSet(x):
    parent[x] = x
    size[x] = 1

find(x):
    if parent[x] != x
        parent[x] = find(parent[x])
    return parent[x]

union(a, b):
    rootA = find(a)
    rootB = find(b)
    if rootA == rootB
        return
    attach smaller root under larger root
    update size of new root

C-Style Sketch

int find(int x) {
    if (parent[x] != x) {
        parent[x] = find(parent[x]);
    }
    return parent[x];
}

void unite(int a, int b) {
    int rootA = find(a);
    int rootB = find(b);
    if (rootA == rootB) return;

    if (size[rootA] < size[rootB]) {
        int temp = rootA;
        rootA = rootB;
        rootB = temp;
    }

    parent[rootB] = rootA;
    size[rootA] += size[rootB];
}

Step Mapping

The animation initializes singleton sets, follows parent links during find, compresses those links when possible, and merges roots only when union sees two different components.

Teaching Note

DSU does not need to scan every element during union. It only touches the representatives and a few parent links, which is why it scales so well.
