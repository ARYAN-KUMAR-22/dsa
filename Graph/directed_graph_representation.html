Current Insight

A directed edge updates only one list direction and only one matrix cell, because the connection goes from source to destination.

What Directed Means

  • A directed edge has an arrow, so A -> B does not automatically mean B -> A.
  • The direction matters in storage, traversal, and matrix updates.
  • That is why the adjacency matrix for a directed graph is not necessarily symmetric.

Adjacency List

  • Store the outgoing neighbors of each vertex.
  • If there is an edge u -> v, add v only to the list of u.
  • This is usually space-efficient for sparse graphs.

Adjacency Matrix

  • matrix[u][v] = 1 means there is a directed edge from u to v.
  • You do not automatically set matrix[v][u].
  • This makes edge lookup fast but uses O(V^2) space.

In-Degree and Out-Degree

  • Out-degree: number of edges leaving a vertex.
  • In-degree: number of edges entering a vertex.
  • Directed graphs care about both because flow has direction.

C Code: Adjacency Matrix

#define V 6

void addEdge(int matrix[V][V], int u, int v) {
    matrix[u][v] = 1;
}

C Code: Adjacency List

void addEdge(Node* graph[], int u, int v) {
    addNode(&graph[u], v);
}
