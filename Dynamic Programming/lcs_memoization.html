How Memoization Works

Cache Before Recursing! We check if LCS(i,j) already exists in memory. If yes, return it instantly (cache hit). If no, compute it and store for future use.

Key Improvement

Polynomial Time!
• Recursion: ~O(2^n) exponential
• Memoization: O(m×n) polynomial
• Cache hits: Dramatic speedup visible in metrics

Top-Down DP

Memoization is "top-down" recursion with caching. Start from big problem, break into smaller ones, cache results. Each (i,j) computed exactly once.
