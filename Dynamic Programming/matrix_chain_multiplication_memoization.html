How Memoization Works

Cache Before Recursing! We check if MCM(i,j) already exists in memory. If yes, return it instantly (cache hit). If no, compute it and store for future use.

Key Improvement

Polynomial Time!
• Recursion: ~O(3^n) exponential
• Memoization: O(n³) polynomial
• Cache hits: Dramatic speedup visible in metrics above

Top-Down DP

Memoization is "top-down" recursion with caching. Start from big problem, break into smaller ones, cache results. Contrast with "bottom-up" tabulation.
