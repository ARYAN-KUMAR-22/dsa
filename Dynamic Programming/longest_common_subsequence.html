What is LCS?

Longest Common Subsequence is the longest sequence of characters that appear in both strings in the same order (but not necessarily consecutive).

Example: LCS("AGGTAB", "GXTXAYB") = "GTAB" (length 4)

How DP Solves It

Build a table dp[i][j] where:
• dp[i][j] = LCS length of first i chars of str1 and first j chars of str2
• If str1[i] == str2[j]: dp[i][j] = dp[i-1][j-1] + 1
• Else: dp[i][j] = max(dp[i-1][j], dp[i][j-1])
