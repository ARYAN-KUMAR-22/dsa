Inorder

Left → Node → Right

  • Visits nodes in sorted order (for BST)
  • Used to: print BST in order, expression evaluation
  • inorder(L); visit(N); inorder(R);
Preorder

Node → Left → Right

  • Root is visited first
  • Used to: copy/serialize a tree, prefix expressions
  • visit(N); preorder(L); preorder(R);
Postorder

Left → Right → Node

  • Root is visited last
  • Used to: delete tree, postfix expressions, folder size
  • postorder(L); postorder(R); visit(N);
Level-Order

Level by Level (BFS)

  • Uses a Queue, not recursion
  • Used to: level-by-level print, shortest path, AVL height
  • queue.push(root); while(!empty) dequeue&enqueue-children

📊 Complexity Comparison

Traversal Data Structure Time Space Use Case
Inorder Call Stack O(n) O(h) Sorted output from BST
Preorder Call Stack O(n) O(h) Tree serialization / copy
Postorder Call Stack O(n) O(h) Deletion / folder sizes
Level-order Queue O(n) O(w) BFS / shortest path

h = height, w = max width of tree
