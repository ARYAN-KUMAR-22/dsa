Heap Properties

  • Complete Binary Tree: filled left to right
  • Min Heap: Parent ≤ Children
  • Max Heap: Parent ≥ Children
  • Stored as an array (index 1..n)

Array Index Rules

  • Root at index 1
  • Parent of i: ⌊i/2⌋
  • Left child: 2i
  • Right child: 2i+1
