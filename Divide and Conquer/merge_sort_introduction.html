Common Mistakes

          

            
Wrong merge bounds
Merging the wrong ranges breaks the sorted invariant even if the recursion is correct.

            
Ignoring extra space
Merge sort is fast, but the classic array version uses O(n) helper space.

            
Forgetting stability
When equal elements appear, taking from the left half first preserves stable ordering.

            
Interview framing
Say "split, recursively sort both halves, then merge" before talking about the recurrence.
