Common Mistakes

          

            
Off-by-one bounds
The update rules for low and high must match the loop condition exactly.

            
Duplicate handling
Finding any match is different from finding the first or last occurrence. The equal case must be handled carefully.

            
Unsorted data
Binary search is correct only when a monotone property or sorted order justifies discarding half the space.

            
Interview framing
State the invariant, the mid calculation, and why one half can be eliminated before discussing the runtime.
